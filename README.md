## Description
The project includes several small games:
- Guess Game
- Memory Game
- Currency Roulette

## Installation
To install the required dependencies, run:
```sh
pip install -r requirements.txt
```

## Usage
To run the main game, use the following command:
```sh
python MainGame.py
```
